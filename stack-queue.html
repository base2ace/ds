📚 Module 3: Stacks (LIFO Data Structure)

Understand the Last-In, First-Out (LIFO) mechanism, real-world analogies, time complexities, and function call stack mechanics in C.

💡 1. What is a Stack?

A Stack is a linear Abstract Data Type (ADT) operating on the LIFO (Last In, First Out) principle. Insertion and deletion occur strictly at a single open endpoint called the TOP of the stack.

🔑 Key Rule: LIFO Access

The element inserted last is always the very first element to be removed. You cannot access lower elements without first popping the elements above them!

🎯 Primary Entry Point: TOP Pointer

All operations (push, pop, peek) are performed at the Top pointer in constant O(1) time complexity.

🌎 Real-World & System Analogies of Stacks

🍽️

A. Stack of Plates

Plates placed in a cafeteria stack. New plates are added to the top (Push) and washed from the top (Pop).

🌐

B. Browser Back Button

Every new URL visited is pushed onto the history stack. Clicking "Back" pops the current URL to reveal the previous page.

↩️

C. Text Editor Undo (Ctrl+Z)

Text modifications are stored in an Undo Stack. Pressing Ctrl+Z pops the most recent edit action.

⚙️

D. C Function Call Stack

CPU reserves Stack Frames for function calls. Returning from a function pops its activation record.

⚙️ C Function Call Stack & Activation Records

When a C program runs, the CPU reserves a Stack Memory Segment. Every time a function is invoked (e.g. main() -> calculateFactorial(5)), a Stack Frame (Activation Record) is pushed onto the stack containing local variables, arguments, and return addresses.

Recursion Call Stack Mechanics Example (Factorial C Code)

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1); // Pushes new Stack Frame onto Call Stack!
}

💻 Essential C Programs for Stacks

Compilable, production-ready C implementations for university exams and coding interviews.

1. Array Implementation of Stack in C (Fixed MAX=5) 

#include <stdio.h>
#include <stdlib.h>
#define MAX 5

int stack[MAX];
int top = -1; // Initial empty state

void push(int val) {
    if (top == MAX - 1) {
        printf("Stack Overflow! Cannot push %d\n", val);
        return;
    }
    stack[++top] = val;
    printf("Pushed %d\n", val);
}

int pop() {
    if (top == -1) {
        printf("Stack Underflow!\n");
        return -1;
    }
    return stack[top--];
}

int main() {
    push(10); push(20); push(30);
    printf("Popped element: %d\n", pop());
    return 0;
}

2. Dynamic Linked List Implementation of Stack in C 

#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node* next;
};

struct Node* top = NULL;

void push(int val) {
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
    newNode->data = val;
    newNode->next = top;
    top = newNode;
}

int pop() {
    if (top == NULL) return -1;
    struct Node* temp = top;
    int val = temp->data;
    top = top->next;
    free(temp);
    return val;
}

int main() {
    push(15); push(30); push(45);
    printf("Popped: %d\n", pop());
    return 0;
}
