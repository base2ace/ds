🔍 Graph Traversals: BFS vs DFS

Graph traversal algorithms visit all vertices systematically:

Breadth-First Search (BFS)

Explores neighbors level-by-level using a Queue. Guarantees finding the shortest path on unweighted graphs.

Depth-First Search (DFS)

Explores as deep as possible along each branch before backtracking using a Stack or recursion.

💻 C Graph Adjacency Matrix Representation Code 

#include <stdio.h>
#define V 5

void addEdge(int matrix[V][V], int i, int j) {
    matrix[i][j] = 1;
    matrix[j][i] = 1;
}

int main() {
    int adjMatrix[V][V] = {0};
    addEdge(adjMatrix, 0, 1);
    addEdge(adjMatrix, 0, 2);
    return 0;
}
