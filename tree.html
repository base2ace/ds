🔄 Depth-First Tree Traversals: Inorder, Preorder, Postorder

Because trees are non-linear data structures, visiting every node requires systematic recursion:

Inorder (Left, Root, Right)

Visits nodes in ascending sorted order on a Binary Search Tree.

Preorder (Root, Left, Right)

Used to copy trees or construct expression tree prefix notation.

Postorder (Left, Right, Root)

Used for deleting trees safely bottom-up without memory leaks.

💻 C Binary Tree Implementation Code 

#include <stdio.h>
#include <stdlib.h>

struct TreeNode {
    int key;
    struct TreeNode* left;
    struct TreeNode* right;
};

struct TreeNode* createNode(int item) {
    struct TreeNode* temp = (struct TreeNode*)malloc(sizeof(struct TreeNode));
    temp->key = item;
    temp->left = temp->right = NULL;
    return temp;
}
